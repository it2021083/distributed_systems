Προαπαιτούμενα: Να έχετε spring boot και maven


Οδηγίες λήψης:


1. Αφού πατήσατε τον σύνδεσμο και είστε στο github(αλλιώς δεν θα διαβάζατε το αρχείο), πατήστε πάνω σε ένα πράσινο κουμπί που θα γράφει ( “ <> code “) και δεξιά από τη λέξη code θα έχει ένα βελάκι προς τα κάτω.
2. Από εκεί πατήστε την επιλογή “Download ZIP” 
3. Έπειτα κάνετε extract το αρχείο στον φάκελο που θέλετε
4. Από εκεί μπαίνετε στον αρχικό κατάλογο της εφαρμογής 
5. Τρέχετε την εντολή mvn spring-boot:run
6. Η εφαρμογή σας πλέον τρέχει στο localhost:3030
7. Ανοίξτε το Postman και κάντε import το json αρχείο που βρίσκεται στο github και με σύμφωνα με το εγχειρίδιο του REST API που είναι επίσης στο gitHub τρέξτε τα api calls.
