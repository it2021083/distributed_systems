1. Τα POST api calls : /api/Admin/employees/update-employee
                        /api/Admin/farmers/update-farmer
                        /api/Admin/users/update-user/{userId}
                        /api/Admin/employees/update-employee
                        /api/Farmer/submitApplication
                        /api/Employee/application/approval/{applicationId}
ενώ εκτελούν σωστά τις λειτουργίες τους, αν πας αμέσως μετά να εκτελέσεις οποιαδήποτε άλλη λειτουργία εμφανίζει error 403 και κάνει null το token.

Σε οποιοδήποτε άλλο Api call το token εξακολουθεί να λειτουργεί κανονικά και να έχει περιεχόμενο εως ότου να παρέλθει ο χρόνος λήξης του.
    
   2. Την τελευταία μέρα και αφού είχαν ολοκληρωθεί οι υπόλοιπες λειτουργίες μέσω Postman έγινε προσπάθεια να φτιάξουμε τις λειτουργίες του διαχειριστή στον browser αλλά συναντήσαμε το εμπόδιο του ότι το token είναι null. Προσπαθήσαμε να εφαρμόσουμε την τεχνική localStorage αλλά απέτυχε μάλλον,επειδή φορτώναμε άλλη σελίδα για κάθε api call(μετά από μικρή, εξαιτίας χρόνου, έρευνα το βρήκαμε ότι ίσως είναι αυτό το θέμα) και έπειτα στραφήκαμε στα  http cookies που δυστυχώς δεν προλάβαμε να ολοκληρώσουμε οπότε θα συνεχίσουμε την ανάπτυξη με Http cookies πιθανότατα με σκοπό να προλάβουμε το τελευταίο παραδοτέο.




   3. Εξαιτίας του προβληματος 2 όπως καταλαβαίνετε η εφαρμογή προς το παρόν τρέχει μόνο σε Postman.
